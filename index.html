K線（蠟燭圖）基礎

K線是技術分析最基礎的工具，每一根K線記錄了特定時間區間內的四個價格：開盤價、收盤價、最高價、最低價。

陽線（上漲）

陰線（下跌）

K線的時間週期

週期用途適合對象
1分 / 5分極短線進出場當沖交易者
15分 / 60分日內波段短線交易者
日K中短期趨勢波段交易者
週K / 月K大趨勢方向長線投資者
核心原則：大週期定方向，小週期找買點。先看週K確認趨勢，再用日K或小時K尋找進場時機。

重要K線型態

單根K線型態

錘子線 (Hammer) 看漲

下影線長度至少為實體的2倍，出現在下跌趨勢底部。代表空方力竭、多方開始反擊，是潛在的反轉訊號。

吊人線 (Hanging Man) 看跌

外型與錘子相同，但出現在上升趨勢頂部。暗示多方力量衰弱，需配合後續K線確認。

十字星 (Doji) 中性

開盤價≈收盤價，形成十字形。代表多空力量均衡，常出現在趨勢轉折點，需觀察後續方向。

大陽線 / 大陰線

實體長、影線短。大陽線代表強烈買盤，大陰線代表強烈賣壓。出現在關鍵位置有突破或崩跌意義。

組合K線型態

看漲吞噬 (Bullish Engulfing) 反轉

看漲吞噬：陽線完全包覆前一根陰線，出現在底部暗示反轉向上。

看跌吞噬 (Bearish Engulfing) 反轉

看跌吞噬：陰線完全包覆前一根陽線，出現在頂部暗示反轉向下。

早晨之星 (Morning Star) 反轉

三根K線組合（陰線→小實體→陽線），出現在底部為強烈看漲反轉訊號。

黃昏之星 (Evening Star) 反轉

三根K線組合（陽線→小實體→陰線），出現在頂部為看跌反轉訊號。

三白兵 (Three White Soldiers) 看漲

三白兵：連續三根陽線，每根收盤價高於前根，代表強勁買盤。

三黑鴉 (Three Black Crows) 看跌

三黑鴉：連續三根陰線逐步走低，代表持續賣壓。

頭肩頂 (Head & Shoulders Top) 反轉

經典反轉型態。由左肩、頭部、右肩構成，跌破頸線確認反轉向下。

頭肩底 (Head & Shoulders Bottom) 反轉

頭肩底則相反，突破頸線確認反轉向上，是底部強烈看漲訊號。

注意：K線型態不是100%準確的預測工具。必須結合成交量、趨勢位置、支撐壓力來綜合判斷，單一型態的勝率通常只有55-65%。

常用技術指標

趨勢類指標

均線 (MA / EMA)

MA5/10：短期趨勢
MA20/60：中期趨勢
MA120/240：長期趨勢

黃金交叉：短均線上穿長均線 → 看漲
死亡交叉：短均線下穿長均線 → 看跌
均線多頭排列（短上長下）= 上升趨勢

MACD

DIF線：快線（12日EMA - 26日EMA）
DEA線：慢線（DIF的9日EMA）
柱狀圖：DIF - DEA

DIF上穿DEA = 黃金交叉（買入訊號）
零軸以上 = 多頭市場
背離：價格創新高但MACD沒有 → 頂背離（危險）

震盪類指標

RSI（相對強弱指標）

衡量漲跌力道的比率，範圍 0-100。

RSI > 70：超買區，可能回檔
RSI < 30：超賣區，可能反彈
RSI 背離：與價格走勢不同步時，為強烈反轉訊號

KD（隨機指標）

K線與D線在 0-100 之間震盪。

K > 80：超買
K < 20：超賣
K上穿D：買入訊號
K下穿D：賣出訊號
在超賣區金叉 = 較可靠買點

量能指標

成交量 (Volume)

價漲量增：上漲有量支撐，趨勢健康
價漲量縮：上漲動能不足，可能見頂
價跌量增：恐慌拋售，可能加速下跌
價跌量縮：賣壓減輕，可能接近底部

「量是價的先行指標」

布林通道 (Bollinger Bands)

由中軌（20日MA）± 2倍標準差構成。

觸及上軌：股價偏高，可能回落
觸及下軌：股價偏低，可能反彈
通道收窄：即將出現大幅波動
通道擴張：趨勢正在發展中

進場策略

進場前的檢查清單

  • 大盤趨勢是否向上或盤整？（不在空頭市場做多）
  • 個股趨勢是否與大盤同向或更強？
  • 是否有明確的技術訊號（型態、指標、支撐）？
  • 成交量是否配合？
  • 風報比是否至少 1:2？
  • 止損位置是否明確且可接受？

五種常見進場方法

1. 支撐位回測買入

股價回落到重要支撐位（前低、均線、趨勢線）且出現止跌K線型態時買入。

優點：止損明確、風報比好
缺點：可能接到下跌刀

2. 突破買入

股價帶量突破重要壓力位（盤整區間上緣、前高、頸線）時買入。

優點：順勢而為
缺點：假突破風險，需確認量能

3. 均線買入法

股價回踩重要均線（如20日、60日）並獲得支撐時買入。均線多頭排列時效果最佳。

條件：均線方向向上 + 縮量回踩 + 止跌訊號

4. 背離買入

股價創新低但 RSI/MACD 未創新低（底背離），代表下跌動能衰竭。

注意：背離可以持續多次，建議等待價格確認反轉再進場

分批進場策略

方式說明適用場景
金字塔加碼首次買入50%，確認方向後加碼30%，再加碼20%趨勢確認後逐步加碼
等分法將資金分成3等份，在不同價位分批買入不確定底部時攤平成本
倒金字塔先少量試探，每跌一段加倍買入有明確價值區間的標的
核心原則：永遠不要一次 All-in。分批進場能降低擇時風險，也讓你在判斷錯誤時損失更小。

止損（停損）設定

鐵律：進場前必須先決定止損位。沒有止損計畫的交易 = 賭博。止損不是認輸，而是保護本金的最重要工具。

常見止損方法

固定百分比止損

跌破買入價 3%-8% 即止損。

優點：簡單明確
缺點：未考慮個股波動性
建議：波動大的股票用較大比例（5-8%），穩定股用較小比例（3-5%）

技術位止損

設在關鍵技術位下方：
- 支撐線下方 1-2%
- 均線下方
- 前一波低點下方

優點：有技術依據，不會被正常波動洗掉
這是最推薦的止損方法

ATR 止損

使用 ATR（平均真實波動幅度）的倍數設止損。

公式：止損價 = 買入價 - 2xATR

優點：自動適應不同股票的波動性
專業交易者常用的方法

移動止損（追蹤止損）

隨著股價上漲，止損位也跟著上移。

方式一：維持在最高價下方 X%
方式二：跟隨均線移動
方式三：跌破前一波回調的低點

「讓利潤奔跑，截斷虧損」

止損的常見錯誤

錯誤後果正確做法
不設止損小虧變大虧，套牢進場前就決定止損位
止損設太近頻繁被洗出場給價格足夠的波動空間
隨意移動止損紀律崩壞只往獲利方向移動，絕不往虧損方向移
心理止損不執行到了止損位卻猶豫使用系統自動止損單
補倉攤平加大虧損部位虧損時減倉，不加倉

如何判斷市場值不值得入場或出場

判斷市場多空的框架

適合入場的條件

  • 大盤站在 MA20 和 MA60 之上
  • 均線呈多頭排列
  • 市場成交量溫和放大
  • 上漲家數 > 下跌家數
  • 領漲族群明確且持續
  • VIX（恐慌指數）處於低位或下降
  • 基本面無系統性風險

應該離場的訊號

  • 大盤跌破 MA60 且均線空頭排列
  • 成交量持續萎縮或恐慌放量
  • 漲停家數銳減、跌停增加
  • 權值股領跌
  • MACD 出現頂背離
  • VIX 飆升至高位
  • 央行升息、黑天鵝事件

多空評分表

每個條件滿足得 1 分，綜合評估：

評估面向多方條件（+1）空方條件（-1）
趨勢大盤在所有均線之上大盤跌破所有均線
量能上漲放量、下跌縮量上漲縮量、下跌放量
廣度超過60%個股上漲超過60%個股下跌
動能MACD 在零軸之上MACD 在零軸之下
情緒VIX < 20VIX > 30
資金外資/法人持續買超外資/法人持續賣超
國際美股/國際市場穩定美股大跌/地緣風險升溫
解讀：
+5 以上 → 積極做多，倉位可到 70-100%
+2 到 +4 → 偏多操作，倉位 40-70%
-1 到 +1 → 觀望或小倉位，倉位 0-30%
-2 以下 → 空手觀望或考慮避險
-5 以下 → 清倉離場

市場週期與策略對應

累積期（底部盤整）

特徵：成交量萎縮、波動縮小、市場冷淡
策略：開始關注、小倉位布局、耐心等待突破

上升期（多頭市場）

特徵：量增價漲、新高不斷、市場樂觀
策略：順勢做多、回調加碼、移動止損保護利潤

派發期（頂部盤整）

特徵：高位震盪、量大但不漲、分歧加大
策略：逐步減倉、收緊止損、不追高

下降期（空頭市場）

特徵：量縮價跌、跌破支撐、市場恐慌
策略：持有現金、不抄底、等待底部確認

基本面分析

基本面分析是透過研究公司的財務數據、經營狀況和產業環境，來評估一檔股票是否值得投資。與技術分析看「價格走勢」不同，基本面看的是「公司本質的價值」。

核心財務指標

EPS（每股盈餘）

公式：淨利 / 流通在外股數

代表公司每一股為股東賺了多少錢。
重點：看趨勢比看絕對值重要。EPS 連續成長 = 公司越來越賺錢。

好 EPS 年增率 > 15%
中 EPS 穩定不衰退
差 EPS 連續下滑或虧損

本益比 (P/E Ratio)

公式：股價 / EPS

代表投資人願意為每1元盈餘支付多少錢，也表示「幾年能回本」。

P/E < 12：可能被低估（或成長性差）
P/E 12-20：合理區間
P/E > 25：可能偏貴（或市場期待高成長）

不同產業的合理 P/E 不同，要跟同業比較

股價淨值比 (P/B Ratio)

公式：股價 / 每股淨值

衡量股價相對於公司帳面價值的倍數。

P/B < 1：股價低於淨值，可能被低估
P/B 1-3：合理範圍
P/B > 5：市場給予高額溢價

適用於資產密集型產業（金融、製造），科技股參考性較低

ROE（股東權益報酬率）

公式：淨利 / 股東權益 x 100%

衡量公司用股東的錢能創造多少報酬，是巴菲特最重視的指標。

優秀 ROE > 15% 且穩定
良好 ROE 10-15%
注意 ROE < 8% 或大幅波動

連續 5 年 ROE > 15% = 護城河指標

財務報表三大表

損益表 (Income Statement)

看公司「賺不賺錢」。

關注重點：
- 營收：是否持續成長？
- 毛利率：是否穩定或提升？（代表競爭力）
- 營業利益率：本業賺錢能力
- 淨利率：最終獲利能力

毛利率下滑 = 產品競爭力可能減弱

資產負債表 (Balance Sheet)

看公司「體質好不好」。

關注重點：
- 負債比：負債 / 總資產，< 60% 較安全
- 流動比：流動資產 / 流動負債，> 1.5 較佳
- 速動比：扣除存貨後，> 1 較安全
- 現金部位：手上現金是否充足

高負債 + 低現金 = 財務風險較高

現金流量表 (Cash Flow)

看公司「真正有沒有錢進來」。

三大現金流：
- 營業現金流：本業帶進的現金（越多越好）
- 投資現金流：買設備、擴廠（負數=在投資）
- 融資現金流：借貸、發股、配息

最佳 營業正 + 投資負 + 融資負
= 本業賺錢、持續投資、還能還債配息

「獲利是意見，現金流是事實」

產業與護城河分析

護城河類型說明例子
品牌優勢消費者願意為品牌付出溢價Apple、Nike、LV
轉換成本客戶換產品的成本很高Microsoft Office、Oracle
網路效應用戶越多、產品越有價值Visa、Meta、Line
成本優勢能以比競爭者更低的成本生產台積電、Costco
法規壁壘牌照、專利、法規保護金融業、藥廠專利
基本面分析檢查清單：
1. 營收與 EPS 是否連續 3 年以上成長？
2. ROE 是否穩定在 15% 以上？
3. 負債比是否低於 60%？
4. 營業現金流是否為正且穩定？
5. 毛利率是否穩定或上升？
6. 公司是否具有明確的護城河？
7. 目前本益比是否在歷史合理範圍內？

長線投資（價值投資）

價值投資的核心理念：以合理或便宜的價格，買入優質公司的股票，長期持有讓複利發揮作用。這是巴菲特、查理蒙格等傳奇投資者的核心策略。

價值投資的核心原則

安全邊際 (Margin of Safety)

只在股價低於內在價值時買入，預留足夠的「安全空間」。

做法：
- 估算公司合理價值
- 在合理價的 7-8 折以下買入
- 折扣越大，安全邊際越高

「用 50 分的價格買 1 元的東西」

能力圈 (Circle of Competence)

只投資你能理解的公司和產業。

問自己：
- 我了解這家公司怎麼賺錢嗎？
- 10 年後這家公司還會存在嗎？
- 我能判斷它的競爭優勢嗎？

「不懂的東西不要碰」

長期持有 (Buy & Hold)

好公司值得長期持有，讓複利效應發揮最大威力。

複利的力量：
- 年化 10%，10 年 → 2.6 倍
- 年化 15%，10 年 → 4.0 倍
- 年化 20%，10 年 → 6.2 倍

「我最喜歡的持有期限是：永遠」— 巴菲特

逆向思考 (Contrarian Thinking)

在市場恐慌時買入，在市場瘋狂時賣出。

關鍵心態：
- 股災是價值投資者的好朋友
- 別人恐懼時貪婪
- 別人貪婪時恐懼

「股市是把錢從急躁的人手中，轉移到有耐心的人手中」

估值方法

本益比估值法

用歷史 P/E 區間判斷目前是便宜還是偏貴。

步驟：
1. 查出過去 5 年的 P/E 區間（例如 10-20）
2. 便宜價 = EPS x 歷史低 P/E
3. 合理價 = EPS x 平均 P/E
4. 昂貴價 = EPS x 歷史高 P/E

買入 股價 < 便宜價
持有 便宜價 ~ 合理價
考慮賣出 股價 > 昂貴價

殖利率估值法

適用於穩定配息的股票。

公式：殖利率 = 現金股利 / 股價 x 100%

便宜價：股利 / 6.25%（= 股利 x 16）
合理價：股利 / 5%（= 股利 x 20）
昂貴價：股利 / 3.33%（= 股利 x 30）

殖利率 > 5% 且配息穩定 = 有吸引力

股利折現模型 (DDM)

用未來股利的現值來估算合理價。

簡化公式 (Gordon Model)：
合理價 = D1 / (r - g)
D1 = 明年預期股利
r = 要求報酬率（如 10%）
g = 股利成長率（如 5%）

例如：股利 5 元，成長率 5%，要求報酬 10%
合理價 = 5.25 / (10%-5%) = $105

自由現金流折現 (DCF)

最嚴謹的估值方式，計算未來現金流的現值。

步驟：
1. 預估未來 5-10 年自由現金流
2. 設定折現率（WACC，通常 8-12%）
3. 計算終值（Terminal Value）
4. 全部折現回今天加總

計算較複雜，可使用下方的長線投資計算器輔助

定期定額 (DCA) 策略

定期定額 (Dollar Cost Averaging)：每月固定投入相同金額，不管股價高低都買。

優點：
- 自動在低價時買多、高價時買少，平均成本
- 不需要擇時，降低心理壓力
- 養成紀律性投資習慣
- 適合沒時間盯盤的上班族

最佳標的：大盤 ETF（如 0050、VOO、SPY）或穩定成長的龍頭股

關鍵：選對標的比擇時更重要。定期定額買到爛公司一樣會虧。

資金管理原則

單筆風險控制

每筆交易最多虧損總資金的 1-2%。

例如：帳戶 100 萬，單筆最多虧損 1-2 萬

計算倉位：
倉位大小 = 可承受虧損 / (買入價 - 止損價)
例：可虧 1萬，止損距離 5元
→ 最多買 2000 股

總體風險控制

最大持倉：不超過 3-5 檔個股
最大虧損：所有持倉同時止損不超過總資金 6%
同產業：同一產業不超過總資金 30%
現金比：永遠保留至少 20-30% 現金

風報比（Risk-Reward Ratio）

公式：風報比 = 預期獲利 / 預期虧損

最低要求：風報比 >= 1:2（虧 1 元要能賺 2 元）
理想目標：風報比 1:3 以上

即使勝率只有 40%，只要風報比維持 1:3，長期仍然能獲利：
10 次交易 → 4 次賺（各賺3）+ 6 次虧（各虧1）= +12 - 6 = 淨賺 6

交易心態管理

克服恐懼

恐懼讓你在該進場時不敢買，在該持有時提早賣。

解法：用合理的倉位降低壓力。如果一筆交易讓你睡不著，代表倉位太重。

控制貪婪

貪婪讓你不設止損、不停利、不斷加碼。

解法：設定明確的目標價和止損價，到了就執行，不帶感情。

避免報復性交易

虧損後急著想翻本，導致衝動下單。

解法：連續虧損 2-3 次後強制休息 1-3 天，清空思緒後重新分析。

建立交易紀律

交易日誌：每筆交易記錄買賣理由、情緒狀態、結果檢討
固定流程：掃描 → 分析 → 計畫 → 執行 → 記錄
定期回顧：每週/每月檢視交易紀錄，找出重複犯的錯誤
