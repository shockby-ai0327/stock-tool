## 1. 市場掃描結果（2026-06 查證）

### 1.1 高端：交易日誌 SaaS 付費意願已被驗證

| 競品 | 定價 | 弱點（= 我們的缺口） |
|---|---|---|
| TradeZella | $29–49/月（年繳 $288–399） | 純英文、無台股、無台灣券商格式 |
| TraderSync | $29.95–79.95/月 | 同上，Elite 級才有回測 |
| Edgewonk | $169–197/年 | 桌面思維、無中文 |
| Tradervue | ~$197/年，有免費層 | 介面老舊、無台股 |

**結論**：交易者願意為日誌工具付月費 $29 以上，市場成立。**全部競品都不支援台股**（券商對帳單格式、三大法人籌碼、中文介面），這是明確的利基缺口。

### 1.2 低端：模板市場有量

- Etsy 上「Notion trading journal」相關商品 **2,000+ 件**，有穩定 bestsellers；Gumroad 同類免費模板曾被領到斷貨。
- 定價帶：US$5–25。
- **用途**：低端模板是「兩週內、近零成本」驗證利基需求的探針，順便賺第一筆。

### 1.3 平行選項：TradingView 指標

- 社群 100,000+ 腳本，invite-only + Whop 收訂閱費是成熟玩法。
- 需遵守 TradingView Vendor Requirements（誠實行銷、不保證獲利、原創工具）。
- repo 的 regime detection／signal 邏輯可包成 Pine Script。**列為 A2 的替代方案**，若模板驗證顯示「台股 journal 需求不足、但指標詢問度高」再切換。

## 3. 主線 A：交易工具產品（階梯式）

### A1 — 交易日誌模板（Week 1–2）

- **產品**：台股+美股雙市場交易日誌模板，Notion 版 + Google Sheets 版，中英雙語。內含：交易紀錄表（含手續費/證交稅欄位）、策略 playbook、勝率/期望值自動統計、心理檢核表、月度復盤模板。
- **差異化**：市面 2,000 件全是英文+美股思維；台股欄位（當沖/隔日沖標記、券商手續費折數、證交稅）沒人做。
- **通路**：Gumroad（國際）+ Etsy（國際）；中文圈用 Gumroad 連結直賣。
- **定價**：US$12（單版）／US$19（雙版 bundle）。
- **我做**：模板全部內容、商品圖、上架文案、SEO 關鍵字、FAQ。
- **你做**：開 Gumroad/Etsy 帳號綁金流（約 1 小時）、按發布、到台股社群（PTT Stock 板、Threads、FB 社團）以個人身分分享。
- **KPI（30 天）**：≥10 筆銷售 → 利基成立，啟動 A2；3–9 筆 → 修價格/文案再測 30 天；<3 筆 → 利基存疑，轉評估 TradingView 指標線。

### A2 — 台股交易日誌 SaaS（Month 1–3，A1 過 Gate 才啟動）

- **產品 MVP 範圍**（刻意砍小）：
  1. 券商對帳單 CSV 匯入（先支援 2–3 家大券商格式）
  2. 自動損益/勝率/期望值統計 + 權益曲線
  3. 每筆交易對照當日三大法人買賣超（repo 現成數據，**獨家功能**）
  4. 中文介面
- **明確不做**（v1）：自動 API 串接券商、回測、行動 App。
- **定價**：NT$199/月 或 NT$1,990/年（約 TradeZella 的 1/4，先搶量）。免費層：手動記錄 20 筆。
- **技術**：Vite 前端（repo 已有 vite-migration 經驗）+ Supabase（auth/DB 免費層）+ 現有 Python 數據管線。
- **我做**：~90% 程式、landing page、文案、Email 流程。
- **你做**：網域購買、Supabase/金流帳號（台灣可用藍新/綠界收 NT$，或 Lemon Squeezy 收美元）、封測期親自回 10 個用戶的訊息。
- **KPI Gate（月 3 末）**：付費用戶 ≥10 → 續投；<10 → 停損，把學到的東西帶去第二個產品（組合策略，不戀棧單一產品）。

## 4. 副線 B：台股籌碼週報（Week 1 啟動，永續低成本運行）

- **內容結構**（全自動生成 + 你過目 10 分鐘）：
  1. 本週大盤狀態（regime detection 輸出）
  2. 三大法人資金流向 Top 10（tw_institutions）
  3. 產業雷達異動（tw_sector_radar）
  4. 新聞情緒儀表（FinBERT 分數彙總）
  5. 策略存活統計專欄（backtest 數據，教育性質）
- **通路**：方格子（中文主場）+ Substack（同步存檔）。
- **頻率**：每週一篇，發布日固定（週日晚），由 cron 產草稿。
- **紅線（必守）**：**不給任何個股買賣建議**——台灣無投顧執照販售個股建議違法。只做數據呈現與教育，每篇附免責聲明。**因此永遠不開收費訂閱的「建議」內容**；週報的商業價值是當 A 線產品的免費獲客漏斗。
- **KPI**：月 3：200 訂閱；月 6：500–1,000 訂閱；週報 → 產品落地頁的點擊率 >5%。

## 8. 誠實的期望值（必讀）

獨立產品的真實分佈：~40% 從未達到月收 US$1K，一半的人月收不到 $1K；做到 $10K MRR 中位數要 12–18 個月。本規劃用「階梯 + Gate + 組合」對抗這個分佈，而不是假裝它不存在。

| 情境 | 機率（粗估） | 月 6 狀態 |
|---|---|---|
| 保守 | ~40% | 模板賣 <10 份、SaaS 沒過 Gate → 6 個月總收入 < NT$5k，但留下週報受眾與第二次出手的市場知識 |
| 基準 | ~40% | 模板月銷 10–30 份 + SaaS 10–50 付費戶 → 月收 NT$5k–30k |
| 樂觀 | ~20% | SaaS 找到 PMF → 月 12 達月收 NT$30k–100k+ |

**三大風險與對策**：
1. **台股 journal 市場太小** → A1 模板用兩週低成本先驗證，不直接蓋 SaaS。
2. **大廠（TradeZella 等）加中文/台股支援** → 護城河押在台股獨有數據整合（三大法人對照），不是押在「中文化」。
3. **你的時間投入斷供** → 每週 3–5 小時是底線（社群發文、客服、決策），這部分我無法代勞——帳號信任必須是真人。

**Kill criteria（白紙黑字）**：A1 30 天 <3 銷售且調整後再 30 天仍 <3 → 棄台股 journal 利基；A2 月 3 末 <10 付費 → 停損。

- **稅務**：境外平台（Gumroad/Etsy/Lemon Squeezy）收入屬個人所得，記得申報；月營收穩定放大後再評估行號/公司與營業稅問題，屆時找會計師。
